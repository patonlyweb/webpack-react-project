# webpack-react-project


## Usage


### Build

Via compose :

    docker-compose build
   
sinon à la main :
       
    docker build -t pperroudon/webpack-react-project

 

### Run

Via compose :

    docker-compose up -d

sinon à la main :
       
    docker run -d -p 80:80 pperroudon/webpack-react-project
